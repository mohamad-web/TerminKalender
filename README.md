# TerminKalender

TerminKalender is a web-based appointment management system developed with **Java and Spring Boot**.  
The application allows users to create, manage, and organize appointments while supporting user authentication, invitations, and role-based access.

This project demonstrates the development of a modern backend application using Spring Boot, REST APIs, database integration, and security mechanisms.

---

## Features

- User registration and authentication
- Create, edit, and delete appointments
- Invite other users to appointments
- Dashboard for managing upcoming events
- Logging system for tracking system activities
- Role-based access control
- REST API endpoints for appointment management
